- Resolution: 1920×1080
- Browser zoom: 90% or 100%
- Run `python start_demo.py`
- Use the supplied `screenshots/overview.png` as the visual target
- Hide browser bookmarks and personal tabs
- Record these windows in order:
  1. Dashboard Overview
  2. Metric Explorer
  3. Pipeline Operations
  4. `db.py` incremental merge logic
  5. `simulator.py` late update/delete generation
  6. Power BI theme, DAX, and CSV folder
